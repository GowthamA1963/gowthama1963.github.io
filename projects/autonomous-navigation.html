Autonomous Navigation Stack

Robust path planning for mobile robots

ROS 2 Humble Nav2 Python/C++ SLAM

Overview

This project focuses on creating a robust navigation stack for a custom mobile robot platform. Using the ROS 2 Navigation 2 (Nav2) stack, the system is capable of autonomous point-to-point navigation, dynamic obstacle avoidance, and real-time mapping.

Key Features

  • Simultaneous Localization and Mapping (SLAM) using SLAM Toolbox.
  • Custom configured costmaps (Global & Local) for efficient path planning.
  • Behavior Tree implementation for complex recovery behaviors.
  • Integrated sensor fusion (LiDAR + IMU) for accurate odometry.

Technical Deep Dive

The core of the navigation stack relies on the `amcl` (Adaptive Monte Carlo Localization) for localization and `dwb_core` for local trajectory planning. I optimized the DWB controller parameters to handle narrow corridors, significantly reducing collision rates by 40% compared to the default configuration.
